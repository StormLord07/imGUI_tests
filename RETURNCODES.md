`Value < 0` indicates that there was an error
`-32,768 < Value < 32,767`

Heavily inspired by Http Error codes.

| Value (`int`) | Description | Type |
|------------------|:-----------:|:-------:|
| 0, 200  | "Operation Succesfull" | OK |
| 304     | "Nothing changed" | OK |
|         |                   |    |
